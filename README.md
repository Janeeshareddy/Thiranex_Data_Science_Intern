# Thiranex_Data_Science_Intern
Data Science Intern at Thiranex
Data Cleaning & Visualization Project

Project Overview

This project focuses on cleaning and analyzing a retail sales dataset to uncover meaningful business insights. The workflow includes handling missing values, removing duplicates, detecting outliers, and visualizing key patterns in the data.

The goal is to transform raw data into meaningful insights using Python.

Dataset

Dataset used: **Sample Superstore Dataset**

This dataset contains sales information including:

* Order details
* Customer segments
* Product categories
* Sales
* Profit
* Regions

 Tools & Libraries Used

* Python
* Pandas (Data Cleaning & Processing)
* Matplotlib (Data Visualization)
* Seaborn (Statistical Visualizations)
* Jupyter Notebook

 Data Cleaning Steps

The following preprocessing steps were performed:

1. Loaded dataset using Pandas
2. Checked dataset structure and summary statistics
3. Identified and handled missing values
4. Removed duplicate records
5. Checked and handled outliers
6. Converted data types where necessary

## 📈 Data Visualization

Several visualizations were created to understand business patterns:

* Sales by Region
* Profit by Category
* Sales distribution
* Correlation heatmap
* Profit vs Sales relationship

These visualizations help identify high performing regions, profitable categories, and business trends.

 Key Insights

Some insights discovered from the analysis:

* Certain regions generate higher sales than others.
* Some product categories contribute more to profit.
* There is a noticeable relationship between sales and profit.
* Data visualization helps reveal trends that are not obvious from raw data.

 Learning Outcomes

Through this project I learned:

* Data cleaning techniques
* Handling missing values and duplicates
* Exploratory Data Analysis (EDA)
* Creating meaningful visualizations
* Extracting insights from raw datasets
